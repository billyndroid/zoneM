## ğŸ¨ Customization

### Colors
The color scheme uses CSS custom properties defined in `:root`:
- Primary: #667eea (Purple-blue)
- Secondary: #764ba2 (Purple)
- Accent: #f093fb (Pink)
- Text: #2d3748 (Dark gray)
